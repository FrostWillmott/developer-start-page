# 🚀 Backend Developer Start Page

A personal start page for Python backend and AI engineers: quick links to RAG/LLM docs, Django/FastAPI, and infra tools, plus copy-paste command cheatsheets.

I built it because I kept reopening the same docs (LangChain, Qdrant, OpenAI, etc.) and retyping the same commands—this puts them one click away as my browser homepage.

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## ✨ Features

- 🔗 Quick links to RAG, LLM APIs, backend frameworks, and infra docs
- 💻 Command cheatsheets for Django, FastAPI, K8s, ports, regex
- 🔍 Search filter across all links
- 🌙 Dark theme, no dependencies

## 📋 Requirements

- Any modern web browser (Chrome, Firefox, Safari, Edge)

## 🚀 Usage

1. Clone and open:
```bash
git clone git@github.com:FrostWillmott/developer-start-page.git
cd developer-start-page
open index.html
```

2. Set as your browser homepage for one-click access.

## 🎨 Customization

Edit `index.html` to adjust links and commands: sections use `<div class="section">`, and the search bar filters by link text.
